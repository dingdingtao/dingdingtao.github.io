Hive四舍五入

2021-03-01 1 min read # Hive
Hive常用四舍五入的方式

大于等于

select ceil(23.33) from table;

=> 24

小于等于

select floor(23.33) from table;

=> 23

四舍五入

select round(23.33) from table;

=> 23

截断

select trunc(23.33) from table;

=> 23

获取符号

select sign(-23.33) from table;

=> -1

百分数实现

select round(20/100, 2) *100 ||'%'  from table

=> 20%

除数取整

select 7 div 3 from table

=> 2
